# Bonho Timeline MVP

App Expo + React Native para configurar momentos de una boda y generar un diagrama de Gantt simple.

## Flujo actual

1. La app abre directo en `Momentos de la boda`.
2. Activas los momentos que quieres incluir.
3. Pones la hora de inicio de cada uno.
4. Presionas `Generar diagrama`.
5. Ves el Gantt del evento.

## Correr en Android Studio

```bash
npm install
npx expo start -c
```

Luego, con el emulador abierto, presiona `a` en la terminal.

## Checks utiles

```bash
npm run typecheck
npm test
npx expo-doctor
```
